# 🚜 Bulldozer Price Prediction using Machine Learning

> Predicting the sale price of bulldozers using historical auction data and advanced Machine Learning techniques.

---

## 📌 Project Overview

This project focuses on building a **robust regression model** to predict the price of bulldozers based on real-world auction data.

It is a **complete end-to-end ML project**, covering everything from raw data to model deployment readiness.

---

## 🎯 Problem Statement

The goal is to accurately predict the **SalePrice** of bulldozers using features like:
- Machine specifications  
- Usage details  
- Auction data  
- Year & model information
