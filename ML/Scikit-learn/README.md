# Scikit-learn ML Models

A modern, hands-on project for building and evaluating machine learning models using **Scikit-learn**.

This repository covers essential steps in the ML pipeline, including data preprocessing, model training, evaluation, and prediction using popular algorithms.

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

## Goal
To provide a simple and practical introduction to machine learning with Scikit-learn.
